# User Directory Table

Simple React app that fetches and displays users from the public API:

`https://reqres.in/api/users`

## Features

- Fetch users from API
- Display in a table with avatar, name, and email
- Search by name or email
- Sort by first name or email (ascending/descending)
- Filter by email domain
- Pagination (handles multiple pages from API)
- Loading spinner while fetching
- Responsive layout for mobile

## Tech Stack

- React (Create React App)
- Plain CSS (no extra UI library)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/user-directory-table.git
cd user-directory-table
```

### 2. Install dependencies
```bash
npm install
```
### 3. Run in development
```bash
npm start
```
App will run on http://localhost:3000.

4. Build for production
```bash
npm run build
```
